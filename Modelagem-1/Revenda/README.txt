# Projeto Revendedora AVENIDA

Enunciado:
AVENIDA é uma famosa revenda de veículos.

Dr. Edecio, fundador e Diretor Geral da empresa, solicitou um sistema que permita cadastrar Carros, Clientes,
Usuários e Propostas.

Obs.:

Os usuários (admins) podem cadastrar os carros.

Os clientes podem fazer propostas para os carros.
